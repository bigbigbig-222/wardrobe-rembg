# REMBG AI抠图 - 快速开始指南

## 🚀 三步快速启动

### 第一步：启动后端服务

**Windows用户**：
1. 打开项目文件夹
2. 双击 `start_rembg_service.bat`
3. 等待看到"Running on http://127.0.0.1:5000"

**macOS/Linux用户**：
```bash
chmod +x start_rembg_service.sh
./start_rembg_service.sh
```

### 第二步：启动网页应用

- 打开浏览器，进入你的电子衣柜网页应用

### 第三步：使用AI抠图

1. 编辑衣服图片
2. 点击工具栏中的"AI抠图"按钮
3. 等待处理完成（10-30秒）
4. 点击"应用到预览"保存

## 🎯 何时使用

### 使用AI抠图
- ✅ 要求更高的抠图质量
- ✅ 背景与衣服颜色差异大
- ✅ 不想手动标记

### 使用手动标记
- ✅ 衣服背景复杂
- ✅ 需要自定义调整
- ✅ 要求最快速度

## ⚙️ 系统要求

- Python 3.8+
- 至少2GB可用硬盘空间（用于模型文件）
- 至少4GB内存
- 可选：NVIDIA GPU（用于加速）

## 💡 常见问题

**Q: 处理很慢怎么办？**
- 确保图片分辨率不超过1000x1000
- 启用GPU加速（如果有NVIDIA GPU）

**Q: 效果不好怎么办？**
- 切换回手动标记模式
- 或者参考REMBG_GUIDE.md中的优化建议

**Q: 服务启动失败？**
- 确认Python版本 >= 3.8
- 查看控制台错误信息
- 参考REMBG_GUIDE.md的故障排除部分

## 🔧 同时使用两种模式

在编辑图片时，你可以：
1. **先用AI抠图** - 获得基础的背景移除
2. **再用手动调整** - 微调边缘或细节

这样结合可以获得最好的效果！
